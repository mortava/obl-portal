# Encompass Developer Connect — Product Overview

ICE's marketing‑facing landing page for Developer Connect. Useful for
positioning, capability summaries, and pointers into the ecosystem.

## Why this matters to the platform

Used for:
- Talking‑points when explaining to a lender what the platform does on top
  of the surface ICE already provides.
- Confirming which capability families are in‑scope for ICE (and which are
  not, so the platform doesn't try to reinvent them).
- Cross‑referencing partner ecosystem (vendors that integrate via Partner
  Connect) when planning third‑party step packs.

## What lives there

- High‑level capability summary: REST APIs for loans, pipeline, contacts,
  documents; API playground; developer guides; community.
- Pointers into Partner Connect, the marketplace path for ISVs.
- Press / case studies for ROI claims in correspondent and wholesale channels.

## Caveats

- This is a marketing surface, not a contract. Treat as **secondary** to the
  developer docs at `developer.icemortgagetechnology.com` for any technical
  decision.
- Capability claims may lag the actual API by a release.
